# Shopify WhatsApp Live Chat Section

A lightweight, speed-optimized Shopify OS 2.0 section for adding a floating WhatsApp live chat button to your Shopify store.

This section is built as a single Liquid file and includes:

- Desktop and mobile settings separately
- Fixed floating WhatsApp button
- Custom WhatsApp number
- Custom desktop and mobile text
- Dynamic current page URL inside WhatsApp message
- Inline SVG WhatsApp icon
- No external CSS file
- No external JavaScript file
- No external image request
- Mobile responsive controls
- Color, spacing, border, shadow, position, and animation settings
- Google PageSpeed friendly code
